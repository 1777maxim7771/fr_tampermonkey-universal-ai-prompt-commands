# Tampermonkey Universal AI Prompt Commands FR

Script Tampermonkey pour remplacer des commandes courtes par des invites IA prêtes à l’emploi dans les chats IA.

Le script remplace uniquement les commandes exactes par des invites préparées. Le texte ordinaire reste inchangé. ChatGPT, Gemini, Claude, Copilot.

## Commandes

- `F1` — Traduis intégralement et fidèlement le texte fourni en français.
- `F2` — Résume le texte fourni en français selon son sens et son contexte.
- `F3` — Rédige en français un résumé thématique très bref de la lettre, sur une seule ligne.
- `F4` — Traduis le texte fourni en allemand simple et clair, niveau A2-B1.
- `F5` — Corrige le texte fourni en français.
- `F6` — Rédige une réponse courte, polie et officielle à cette lettre en allemand simple, niveau A2-B1.
- `F7` — Explique en français simple ce que signifie ce texte.
- `F8` — Extrais tous les faits importants du texte et structure-les en français : personnes, organisations, adresses, dates, délais, montants, numéros de documents, demandes, décisions, obligations, documents et prochaines étapes.
- `F9` — Établis en français une liste claire des actions à effectuer à partir de ce texte.
- `F10` — Rédige à partir du texte fourni une lettre officielle, polie et claire en allemand, niveau A2-B1.

## Installation

Installez Tampermonkey, ouvrez le lien du userscript, confirmez l’installation puis saisissez la première commande dans un chat IA.

[Userscript](https://raw.githubusercontent.com/1777maxim7771/fr_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js)

Le script s’installe dans l’extension Tampermonkey, pas dans GitHub ni dans un site. GitHub stocke uniquement le fichier.

Tampermonkey → Dashboard → Utilities → Import from URL:

`https://raw.githubusercontent.com/1777maxim7771/fr_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js`
